Adding simple emoji reactions to your blog posts.

== Description ==

Adding simple emoji reactions to your blog posts.

Informations are saved in the post meta.

Today no configurable options, will come in future.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `simple-emoji-reactions` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Done. Visit your blog post single view.
